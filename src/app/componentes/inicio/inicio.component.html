¿Como funciona?

Asignación de idioma predeterminado:

  1. Idioma almacenada en localStorage
  2. Idioma en la configuración del idioma del navegador, si es español ('es'), caso contrario va en Ingles ('es') por defecto
    (Imaginemos que entra un portugues o un italiano)

OBSERVACIONES

  • SHIFT + CTRL + J // para ver la consola en 'aplication' como se guarda en localstorage
  • por defecto esta en ingles ('en') y la opcion que guardes persiste, fijate actualizando con F5
  • Para hacer pruebas de cambio de idioma... chrome://settings/languages#lang

Paquetes NPM

commands used for package installation
  • npm install @ngx-translate/core --save
  • npm install @ngx-translate/http-loader --save
